# PRODIGY_DS_03
🌳 Bank Marketing Prediction using Decision Tree Classifier
📌 Project Overview

This project was developed as part of Prodigy InfoTech Data Science Internship – Task 03.

The objective is to build a Decision Tree Classifier capable of predicting whether a customer will subscribe to a bank term deposit based on demographic and behavioral information collected during previous marketing campaigns.

The project demonstrates the complete machine learning workflow, including:
Data preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Decision Tree Classification
Model Evaluation
Data Visualization
Business Insights

Using Machine Learning, we can predict customer responses and help banks:
Improve campaign efficiency
Reduce marketing costs
Increase conversion rates
Identify high-value customers

📂 Dataset
Dataset Name: Bank Marketing Dataset
Source: UCI Machine Learning Repository
This dataset contains customer demographic information and campaign-related details collected from direct marketing campaigns of a Portuguese banking institution.

📈 Exploratory Data Analysis
The project performs detailed EDA including:

🌳 Decision Tree Model
The Decision Tree Classifier learns decision rules from customer information and predicts whether a customer is likely to subscribe to a bank term deposit.
The project compares different splitting criteria such as:
Gini Index
Entropy
